# K-Management website

Jekyll-based GitHub Pages site for K-Management Limited.

## Structure

- `_layouts/default.html` - shared page shell
- `_includes/*.html` - editable content blocks
- `assets/css/style.css` - styling
- `assets/js/site.js` - small interactions
- `index.html` - homepage
- `tariffs.html` - indicative tariffs and engagement page
